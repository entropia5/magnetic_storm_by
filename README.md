# Geomagnetic Telegram Bot (Республика Беларусь)

Это telegram бот на C++ для мониторинга геомагнитной активности (индекс Kp) и прогнозирования магнитных бурь. Специально разработан для жителей Беларуси и полезен пользователям по всему миру при внесении некоторых изменений в исходный файл geomagnetic_bot_by.cpp.

Основные возможности:

1. Автоматическая рассылка: Ежедневный подробный прогноз на 24 часа каждое утро в 09:00.
2. Текущее состояние: Мгновенная проверка индекса Kp в режиме реального времени.
3. Интеллектуальный интерфейс.
4. База данных: Бот сохраняет список подписчиков в файл, что позволяет не терять аудиторию после перезагрузки сервера.



Стек:

1. C++17
2. Библиотеки: [CPR](https://github.com/libcpr/cpr) — для HTTP-запросов к Telegram и NOAA API...И [nlohmann/json](https://github.com/nlohmann/json) — для парсинга JSON данных.
3. Источник данных: Национальное управление океанических и атмосферных исследований (NOAA, USA).
